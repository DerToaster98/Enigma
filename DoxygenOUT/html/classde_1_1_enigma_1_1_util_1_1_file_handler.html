Ausführliche Beschreibung

FileHandler, der die Funktionalitäten mit den Dateien inne hat.

Autor
Nikolai Klatt

Realisiert als Singleton, das heißt nur eine Instanz pro Programminstanz.
Zugriff über getFileHandler() Methode von überall aus möglich.

◆ getFileHandler()

static FileHandler de.Enigma.Util.FileHandler.getFileHandler ( )
static

Methode zum Erstellen des Singelton FileHandler-Objektes.

Rückgabe
gibt das fileHandler Singelton Objekt zurück

Erstellt ein Singleton Objekt, sobald die Instanz benötigt wird und gibt, falls das Objekt schon existiert das Objekt zurück. Das stellt sicher, dass es nur einen FileHandler gibt und dieser von allen Objekten von überall aus genutzt werden kann.
