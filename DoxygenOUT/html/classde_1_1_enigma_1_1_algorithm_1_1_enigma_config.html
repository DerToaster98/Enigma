void de.Enigma.Algorithm.EnigmaConfig.checkMills ( )

Diese Methode überprüft die Walzen, und zwar, ob sie drehen sollten.

Überprüft die Walzen auf ihre Kerben. wenn die dritte Walze die Kerbe überschritten hat, so rotiert auch die zweite.
Hat auch die zweite Walze die Kerbe überschritten, so rotiert auch die erste....
