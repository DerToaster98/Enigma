Ausführliche Beschreibung

Klasse, um eine Enigma Datei zu erzeugen.

Autor
Nikolai Klatt

Die Enigma Datei wird benötigt, um die Daten für das System einfach zugänglich zu speichern. So können wir sie zu einem späteren Zeitpunkt wieder einlesen lassen. Die Datei soll als Json Datei formatiert sein.
